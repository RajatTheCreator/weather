# 🌤️ Weather App

A modern and responsive weather application that provides real-time weather updates with smooth animations and an intuitive user experience.

## ✨ Features

- 🌍 Current location weather
- 🔍 Search weather by city
- 📊 Detailed weather information
- 🌡️ Temperature, humidity, wind speed
- 🌅 Sunrise & sunset
- 🎨 Dynamic weather animations
- 📱 Fully responsive design

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Weather API
